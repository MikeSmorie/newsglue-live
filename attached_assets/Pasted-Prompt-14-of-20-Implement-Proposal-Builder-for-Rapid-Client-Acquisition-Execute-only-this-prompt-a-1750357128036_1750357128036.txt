Execute only this prompt and don’t try to interpret or anticipate what comes next.

Purpose:
Enable lightning-fast generation of professional strategic proposals using scraped campaign data, live newsjack outputs, and campaign metadata.

⸻

Scope:
	•	Generate proposals from a single client name input
	•	Include real campaign data and newsjacked content
	•	Match formatting of in-code-we-trust-2025-06-19.pdf
	•	Embed company logo, brand details, and page structure
	•	Export to PDF, HTML, Word (.docx), and Rich Text

⸻

⸻

→ Fetches: campaign name, website URL, CTA, tone, platforms, emotional hooks
